# 📚 StudySync

> Find your perfect study match — built for FindU x Raikes Hacks 2026

## What it does
StudySync matches students with study partners in their exact courses. Join a live session room with built-in chat, a shared Pomodoro timer, and a goal checklist — all in one place.

## Features
- 🎯 Smart course-based matching
- 💬 Live session chat
- ⏱ Shared Pomodoro focus timer
- ✅ Session goal checklist  
- 🌙 Light & dark mode

## Built with
React · Vite

## Run locally
cd studysync
npm install
npm run dev
